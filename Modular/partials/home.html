Metabolic In Silico Network Expansion Databases

Welcome to the home of the MINE databases. These databases expand the known metabolome using the BNICE framework with hand-curated reaction rules. If it's your first time on the website, we recommend taking one of the tours below. Access to the database is also possible through client APIs in JavaScript, Python and Perl. Please contact us for the latest versions.
