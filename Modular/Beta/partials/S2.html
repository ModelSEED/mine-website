Table S2: Chemical Damage Reaction Rules

Notes on the table:
  1. Clicking on the button in the ‘Basis’ column links to the reaction in Table S1 on which the Reaction Rule is based.
  2. The subscript near atoms describe the context of the atom. The letters indicate the following:
    • W = Wildcard (Any context)
    • 0 = No ring structure (default)
    • B = benzene (six member aromatic ring with any functional groups coming off)
    • N = nonbenzene ring (any ring that's not a six-member aromatic ring)
    • 6 = any six member ring
    • 5 = any five member ring
    • 4 = any four member ring
    • L = Large ring (any ring larger than 6 atoms)
    • H = any heteroaromatic ring
    • F = any fused ring
    The numbers indicate the number of that type of ring the atom participates in with 9 being any number including 0.
