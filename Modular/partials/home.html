Metabolic In Silico Network Expansion Databases

Welcome to the home of Metabolic In silico Network Expanstions. These databases expand the known metabolome using the BNICE framework with hand-curated reaction rules. There are a variety of methods to interact with the database: Search for compound names or identifiers names with quick search, find similar chemicals with structure search, or annotate accurate masses from the metabolomics. Access to the database is also possible through client APIs in JavaScript, Python and Perl. Please contact us for the latest versions.
